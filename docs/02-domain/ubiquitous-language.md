# Ubiquitous Language

## Terms

### SkyRoute

The travel aggregator platform being built in this challenge.

### Traveler

The end user who searches for flights and confirms a booking.

### Provider

An airline source integrated into SkyRoute that supplies flight offers.

### Flight Search Criteria

The full set of user inputs required to search for flights:

- origin
- destination
- departure date
- passenger count
- cabin class

### Flight Offer

A returned, comparable flight option from a provider.

### Base Fare

The starting price before provider-specific pricing adjustments are applied.

### Price Breakdown

The representation of per-passenger price, passenger count, and total price for a flight offer or booking summary.

### Route

The combination of origin airport and destination airport for a flight.

### Domestic Route

A route whose origin and destination are in the same country.

### International Route

A route whose origin and destination are in different countries.

### Passenger Details

The booking form data entered by the traveler:

- full name
- email address
- document number

### Document Number

The route-dependent traveler document value captured during booking.

### Booking

The act of confirming a selected flight offer.

### Booking Confirmation

The successful result returned by the backend after booking, including a booking reference code.
